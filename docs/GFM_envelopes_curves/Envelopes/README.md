# GFM Conformity Assessment - Envelopes

This directory contains a set of examples envelopes for GFM conformity assessment.
They correspond to the envelopes obtained by using the following parameters for the voltage source behind impedance equivalent representation of the installation:

* Nominal power: 100 MVA,
* Nominal voltage: 225 kV,
* Min and max active power: -/+ 120 MW
* Min and max reactive power: -/+ 30 MVAr
* Damping factor D: 250 (overdamped), 95 (underdamped)
* Inertia constant H: 5,
* Effective reactance Xeff: 0.27 pu

The envelopes corresponding to the different test cases are then available in the subdirectories.
